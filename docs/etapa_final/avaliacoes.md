## Introdução
Este documento apresenta uma síntese das avaliações dos protótipos desenvolvidos no projeto. A análise foi realizada por meio da combinação de dados, incluindo entrevistadores, número de participantes e os membros do grupo que participaram das avaliações. Diversas avaliações foram realizadas, conforme detalhado na Tabela 2.

## Objetivo
O objetivo deste documento é relatar as avaliações dos protótipos desenvolvidos, fornecendo uma síntese detalhada das mesmas. Este artefato foi elaborado pela integrante [Renata Quadros](https://github.com/Renatinha28).

## Metodologia
Para cada avaliação, utilizou-se uma metodologia específica, como entrevistas, testes de usabilidade e questionários. As metodologias aplicadas são descritas na Tabela 2. Cada integrante ficou responsável por avaliar no minímo uma funcionalidade, como descrito na Tabela 1.

Por meio dessas avaliações, foi possível obter uma compreensão mais aprofundada do desempenho e da usabilidade dos protótipos e modelos conceituais desenvolvidos. As avaliações foram fundamentais para identificar pontos fortes e áreas que necessitavam de melhorias nos protótipos e modelos conceituais. Esses insights valiosos serviram como base para realizar ajustes necessários nos protótipos desenvolvidos.

Além disso, as avaliações foram extremamente importantes para a equipe, permitindo um contato direto com os usuários e a aplicação prática dos métodos de "Entrevista" e "Teste de Usabilidade". Em resumo, as avaliações realizadas nos protótipos e modelos conceituais forneceram informações essenciais e contribuíram significativamente para a evolução do projeto.
